WaveNet Modeling of Distortion Pedal Using Spectral Features

This web site is our simulation results for the Ibanez SD9.
Method
Method MSE: The conventional method that uses only the mean square error in the time domain.
Method Conv. [11]: The conventional method that uses only the error to signal ratio with high-pass filter in the time domain.
Method PS-EUC: The proposed method that uses MSE in time domain and Euclidian distance in frequency domain.
Method MFS-KL: The proposed method that uses MSE in time domain and generalied KL divergence in frequency domain.
